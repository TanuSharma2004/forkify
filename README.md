Live link : https://forkify-tanu-sharma.netlify.app/

🚀 Key Features: 
🔍 Recipe Search: Look up any keyword (like "pizza" or "chicken") to fetch a list of recipes from an API with loading spinners while data downloads.
👥 Dynamic Servings Changer: Clicking the plus or minus buttons updates the number of servings and recalculates every ingredient quantity automatically.
🔖 Saved Bookmarks: Users can bookmark any recipe they like. The app saves these bookmarks directly inside the browser's Local Storage so they don't disappear when you refresh.
📤 Add Your Own Recipes: Includes a form where you can submit your own recipes with specific formatting constraints, which are then saved to the API database.
📑 Pagination Buttons: Divides search results into clean pages ($10$ recipes per page) so the user isn't overwhelmed by one massive scrolling list.
